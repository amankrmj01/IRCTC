# IRCTC Railway Booking System

This project starts as a basic Java application and will gradually evolve into a complete, multithreaded Spring Boot
system. It is currently a simple command-line Java application for booking railway tickets, viewing trains, and managing
users.

## Features

- User registration and login
- View available trains and details
- Book tickets with seat selection
- Data stored in JSON files

## Requirements

- Java 21 or higher
- Gradle (or use the included wrapper)

## How to Run

1. Build the project:
    - On Linux/macOS: `./gradlew build`
    - On Windows: `gradlew.bat build`
2. Run the application:
    - On Linux/macOS: `./gradlew run`
    - On Windows: `gradlew.bat run`

## Project Structure

- `src/main/java/irctc/` - Source code
- `src/main/resources/` - Data files (JSON)
- `build.gradle.kts` - Build configuration
